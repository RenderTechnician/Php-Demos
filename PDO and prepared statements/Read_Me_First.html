Step 1)

Step 2)
Once Xampp has finished installing open the Xampp Control Panel and make sure the Apache and MySQL services are running



SQL [Creating The Database In Commpand Prompt]

Step 1)
Open command prompt or type 'CMD.exe' into the start menu and hit enter




Step 2)
Type 'C:\xampp\mysql\bin\mysql -u root -p', when it prompts for a password press enter.
Your Command Prompt window should now look like the image below




Step 3)
Type the following into Command Prompt. Or copy and paste each line individually (hint: it's quicker if you do this)


CREATE DATABASE student;

USE student;

CREATE TABLE students(

SID VARCHAR(128),

Name VARCHAR(128),

Email VARCHAR(128)) ENGINE MyISAM;



At this point the database has been created and you can use the page here to add values to the database.
